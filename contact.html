Frequently Asked Questions

How is accuracy calculated? +

Accuracy is determined by subtracting missed keystrokes from total keys typed, divided by the total keys typed.

Are sounds mandatory? +

No. Sounds are enabled by default after interaction, but you can disable them anytime from the Settings page.

Can I practice code syntax? +

Currently we focus on common English words. Code typing practice modes will be added in future updates.
